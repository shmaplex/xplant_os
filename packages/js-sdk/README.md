# @shmaplex/xplant-sdk

> The SDK source has moved to its own dedicated repository:
> **[github.com/shmaplex/xplant-sdk](https://github.com/shmaplex/xplant-sdk)**

Install the published package:

```bash
npm install @shmaplex/xplant-sdk
```

Then follow the [quickstart guide](https://github.com/shmaplex/xplant-sdk#quick-start)
in the SDK repo for full usage, type reference, and contribution instructions.

---

> **Note on package scope:** Published as `@shmaplex/xplant-sdk` while the `@xplant` npm
> org is being arranged. Once available, the canonical name will become `@xplant/sdk` with
> a deprecation notice on this package — no import or API changes required.
